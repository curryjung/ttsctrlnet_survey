🗣️ TTS 감정 조절 연구를 위한 음성 평가 설문

안녕하세요. 본 설문은 다양한 TTS (Text-to-Speech) 시스템에서 생성된 음성 샘플을 평가하여, 감정 표현 및 음질의 품질을 비교 분석하기 위한 목적을 가지고 있습니다.

각 음성 샘플은 총 7가지 모델로부터 생성되었으며, 다음 2개의 데이터셋 기반 문장으로 제작되었습니다:

  • Part 1: 참조 음성 11개, 참조 음성: 일본어 / 결과 음성: 영어
  • Part 2: 참조 음성 8개, 참조 음성 및 결과 음성: 영어

✅ 설문 진행 방법

각 샘플을 들은 후, 아래의 3가지 항목에 대해 1점(매우 나쁨 / 비슷하지 않음) ~ 5점(매우 좋음 / 매우 비슷함)으로 평가해 주세요.

※ 모델명(A, B, C 등)은 항목마다 랜덤으로 제시됩니다.

📌 평가 항목

  1. 목소리 유사도
    기준 음성과 비교했을 때, 생성된 음성이 얼마나 같은 사람의 목소리처럼 들리는지 평가해 주세요.
    예: 발화 톤, 억양, 음색의 유사성
    1점: 전혀 다른 사람 같음 / 5점: 완전히 동일한 화자 같음

  2. 자연스러움
    생성된 음성이 얼마나 자연스럽게 들리는지 평가해 주세요.
    예: 목소리가 너무 단조로운지
    1점: 매우 부자연스러움 / 5점: 사람의 발화처럼 자연스러움

  3. 감정 표현 유사도
    기준 음성과 비교했을 때, 생성된 음성이 같은 감정을 얼마나 잘 표현하고 있는지 평가해 주세요.
    예: 기쁨, 분노, 슬픔 등의 감정이 잘 전달되는지
    참조음성에서 차례대로 등장한 음성의 감정 순서 와 동일한 순서로 각 모델들이 음성을 생성했는지도 중요합니다.
    Part 2 에서는 하나의 참조 음성 내 두개의 감정이 차례대로 등장하므로, 각 모델들이 각 감정을 차례대로 잘 표현했는지 특히 중요합니다.
    참조음성과 결과물을 끝까지 듣고 비교 하여 각 항목에 대한 점수를 매겨주세요.
    1점: 감정이 잘 전달되지 않음 / 5점: 감정이 시간대 별로 명확하게 잘 전달됨

🚨 주의사항

  • 긴 설문에 응해주셔서 감사합니다. 약 30분 이상 소요될 예정입니다. 중간에 결과가 저장되지 않으니 설문도중 결과를 잃어버리지 않도록 유의해주세요.
  • 또한 모든 항목을 마무리해야 제출가능합니다. 제출버튼을 누르면 제출완료 페이지가 나오며 제출 버튼을 여러번 반복해서 누르지 않도록 유의해주세요.
  • 테스트를 위해서 실제 설문조사하기 전, 아래 "모든 항목 체크" 버튼을 누르면 모든 항목이 5점으로 체크됩니다. 테스트 목적으로만 활용해주세요.

Part 2: 감정 변화 평가

  1. 목소리 유사도
    생성된 음성이 기준 음성과 얼마나 같은 사람처럼 들리는지 평가합니다.
    예: 발화 톤, 억양, 음색의 유사성
    1점: 전혀 다른 사람 같음 / 5점: 완전히 동일한 화자 같음
  2. 자연스러움
    생성 음성이 얼마나 자연스럽게 들리는지 평가합니다.
    1점: 매우 부자연스러움 / 5점: 사람의 발화처럼 자연스러움
  3. 감정 표현 유사도
    기준 음성과 비교해 생성된 음성이 두 개의 감정을 순서대로 얼마나 잘 표현하는지 평가합니다.
    - 감정의 구별 여부
    - 각 감정의 전달력
    - 참조 음성과 동일한 감정 순서로 표현되었는지

    1점: 감정이 구별되지 않고 하나도 전달되지 않음
    2점: 감정이 구별되지 않거나 하나만 전달됨
    3점: 두 감정이 구별되나 전달력은 약함
    4점: 두 감정이 구별되고 하나는 정확히 전달됨
    5점: 두 감정 모두 명확히 구별되고 정확히 전달됨
